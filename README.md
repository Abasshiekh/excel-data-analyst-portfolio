## Featured Project: Interactive Sales Dashboard

![dashboard preview](./09-dashboard/dashboard.png)

**Problem:** Sales leadership needed a single screen to check revenue, margin, and top regions without opening a raw spreadsheet.
**Approach:** Built a pivot-backed dashboard with 4 KPI tiles (Revenue, Units, AOV, Margin %), a monthly trend line, a category bar chart, and a region slicer + date timeline wired to both charts.
**Result:** Reduced a 15-minute manual weekly report to a 10-second refresh + glance.
